How it works

1

Create your session

Add polls, quizzes, and short text questions in the browser. No files to edit.

2

Share the QR code

A unique room code and QR code are generated automatically. Show it on screen and students join instantly.

3

Present live

Step through your activities, see answers update in realtime, and reveal correct answers when ready.

📊

Live polls

Ask a question with multiple options. Answers update live while you present.

🧠

Quizzes

Add a correct answer and reveal it when ready. Great for checking understanding.

💬

Short text responses

Collect open-ended answers. Students can submit short written responses on their phones.

📱

QR code join

Students scan a QR code or type a short link. No app install, no sign-up.

🔓
