Markdown for live interactive teaching

SeminarSmack

A minimal browser companion for live polling, short text prompts, quizzes, and embeddable Marp results. Deploy once, define sessions as JSON, and run everything from GitHub Pages with Supabase Realtime.

Preview presenter page Preview participant page Preview embed page

Presenter

Drive the room

Load a JSON session, step between activities, reset responses, lock submissions, reveal quiz answers, and share join or embed links.

Participant

Join from any browser

Audience members open a simple URL, answer the current prompt, and stay in sync with the presenter without any account or install.

Embed

Drop into Marp

Use the read-only embed URL in an iframe to show live results inside slides, speaker notes, or a projector view.

Quick Start

Build launch links

Session files

Author content as JSON

Put session files in /docs/sessions/ and load them with ?session=filename. The app stays static while the content remains easy to version, fork, and reuse.

Deployment

Ship via GitHub Pages

The included GitHub Actions workflow injects Supabase config into docs/config.js during deployment so forks can bring their own project URL and publishable key.
